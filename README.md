- Real-time multiplayer experience
- Auto-daub numbers
- Multiple ticket support

## Tech Stack

- **Frontend:** React + Vite
- **Styling:** TailwindCSS
- **State Management:** React Context/Redux
- **Backend:** Supabase (Coming soon)
- **Payments:** UPI Integration (Coming soon)
